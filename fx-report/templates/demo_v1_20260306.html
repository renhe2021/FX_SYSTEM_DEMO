03为什么做这个

需求端 — 客户在要

跨境支付的竞争已经从通道竞争走向服务竞争。客户（东南亚 PSP、银行、电商平台）希望合作方能提供 FX 领域的专业洞察，不只是换汇管道。我们收到的走廊分析、汇率研判的咨询越来越多，但缺少标准化产品来响应。

供给端 — 我们有条件

我们已有 5 个数据源覆盖 24 个币对（ECB、Bloomberg、ExchangeRate-API、Alpha Vantage、Open Exchange），有团队沉淀的东南亚各市场分析框架，有客户画像和走廊配置。AI（Perplexity + Claude）提供了把这些串起来的能力——按走廊搜索新闻、生成市场评论、标注 sentiment，几分钟出一份完整研报。

商业端 — 可以变现

研报产品天然支持分级定价：

  • 基础版（月报、核心走廊）— 增值服务，强化客户关系
  • 高级版（周报 / 日报、定制分析、对冲建议）— 独立变现
  • 同时建立 Tenpay Global 在 FX 领域的品牌专业度

04核心产品：研报平台

产品架构

使用流程（以越南 PSP 为例）

1 选择客户

点击「Vietnam PSP」，系统自动加载客户画像：走廊 USD→VND、CNH→VND，月交易量 $50M，关注央行干预和侨汇流入。

客户卡片区 — 点击选中，画像自动加载到右侧面板
2 勾选报告章节 + 配置 AI

勾选需要的章节（汇率概览、走廊分析、波动率、新闻等），可选输入 Perplexity / Claude API Key 启用 AI 新闻搜索和执行摘要自动生成。

实时嵌入 — 研报平台完整界面（可交互）
3 AI 新闻搜索 + 评论生成

一键搜索 VND 相关实时新闻，AI 结合新闻、汇率数据和客户画像生成市场评论。评论内容可在界面上直接编辑后再出报告。

AI 引擎：Perplexity Sonar 搜索实时新闻 + Claude Sonnet 生成执行摘要，按客户走廊和关注点定制化搜索 prompt。
4 一键生成报告

产出一份聚焦 VND 走廊的投行级研报，包含实时汇率表格、波动率图表、AI 新闻解读和市场展望，可直接发送给客户。

试一下：换一个马来银行客户，同样的流程，输出完全不同 — 走廊变为 USD→MYR，新闻聚焦 BNM 利率政策和棕榈油出口数据。
