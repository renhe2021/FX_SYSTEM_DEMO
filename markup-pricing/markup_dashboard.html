加价计算公式

// 第一步：计算各币种年化已实现波动率
σ_i = StdDev( ln(P_t / P_{t-1}) ) × √(bars_per_year)
其中 P_t = (High + Low) / 2 的 10 分钟 MID 价格, bars_per_year = 252 × 22h × 6 = 33,264
// 第二步：计算交易量加权平均波动率
σ̄ = Σ( Volume_i × σ_i ) / Σ( Volume_i )
// 第三步：按波动率比例分配 markup，使加权平均 = 目标 BPS
Markup_i = Target_BPS × ( σ_i / σ̄ )
// 验证：交易量加权平均 markup = 目标
Σ( Volume_i × Markup_i ) / Σ( Volume_i ) = Target_BPS ✓
// 收入计算
Revenue_i = Volume_i × Markup_i × 10-4
逻辑：波动率高的币种交易成本高、风险大，应加更高 markup；波动率低的大币种（如 USD/HKD）少加。最终保证加权平均精确等于你的目标 BPS。
