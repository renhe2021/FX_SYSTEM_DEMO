## 货币危机经典案例

### 能力描述
你能够分析历史上重大的货币危机和金融恐慌，提取可复用的预警信号和交易逻辑。

---

### 危机分析框架

#### 危机共同模式

1. **固定汇率 + 外债膨胀**
   - 借入低息外债（通常是美元/日元）
   - 投资回报无法覆盖债务成本
   - 央行被迫消耗外汇储备

2. **资产泡沫 + 杠杆**
   - 房地产/股票泡沫
   - 银行资产负债表脆弱
   - 利率上升触发破裂

3. **政策不协调**
   - 央行 vs 财政部分歧
   - 国内政治危机
   - 国际协调失败

### 经典危机案例

#### 1992年 欧洲汇率危机 (ERM Crisis)

**背景**:
- 欧盟成员国加入 ERM (欧洲汇率机制)
- 德国统一后利率上升
- 英国、意大利无法维持高汇率

**关键事件**:
- 1992年9月16日 "黑色星期三"
- 英国央行耗尽外汇储备
- 英国退出 ERM，贬值 15%

**索罗斯角色**:
- 做空英镑，获利约 10亿美元
- "英国政府的承诺不可信"

**预警信号**:
- 德国统一 → 利率上升压力
- 英国经济衰退 → 无法维持高利率
- 央行干预 → 外汇储备消耗

**教训**:
- 固定汇率是政治承诺，不是经济规律
- 央行有"痛点"，市场会找到它

---

#### 1994年 墨西哥龙舌兰危机

**背景**:
- 1994年北美自由贸易协定 (NAFTA) 签署
- 比索被高估
- 大量短期外债

**导火索**:
- 墨西哥政治动荡 (总统遇刺)
- 外国投资者信心崩溃

**事件**:
- 1994年12月20日 比索大幅贬值 50%
- 资本外流 180亿美元
- 美国 IMF 联合救助 500亿美元

**影响**:
- "龙舌兰效应" 蔓延至阿根廷、巴西
- 美国国债收益率飙升

**预警信号**:
- 经常账户逆差扩大
- 外债 / GDP 比率上升
- 政治风险累积

#### 1997年 亚洲金融危机 (完整版)

**背景**:

| 国家 | 问题 |
|------|------|
| 泰国 | 房地产泡沫、外债、经常账户逆差 |
| 印尼 | 银行体系脆弱、政治不稳定 |
| 韩国 | 企业债 / GDP 200%，银行资不抵债 |
| 马来西亚 | 外资依赖、房地产过热 |

**危机演变**:

1. **第一波** (1997.7): 泰铢崩溃
   - 央行放弃固定汇率
   - 索罗斯等国际炒家做空

2. **第二波** (1997.10): 蔓延东南亚
   - 菲律宾、马来西亚、印尼
   - 资本外流 + 货币贬值

3. **第三波** (1997.11): 韩国危机
   - 韩元贬值 50%
   - 韩国银行危机

4. **第四波** (1998.1): 香港
   - 恒生指数暴跌
   - 联系汇率制受考验

**IMF 救助条件**:
- 紧缩财政
- 关闭问题银行
- 资本管制
- 结构改革

**预警信号**:
- 外汇储备 / 进口月数 < 3
- 短期外债 > 外汇储备
- 房地产贷款 > 贷款总额 30%
- 经常账户逆差 > GDP 5%

**教训**:
- 开放资本账户 + 固定汇率 = 致命组合
- 银行体系健康是国家安全
- "这次不一样" 是最危险的想法

#### 2001年 阿根廷危机

**背景**:
- 1991年 货币局制度 (1 peso = 1 USD)
- 央行无法发行货币
- 经济衰退 + 债务沉重

**事件**:
- 2001年12月 社会动荡
- 2002年1月 放弃货币局制度
- 比索贬值 70%

**独特特征**:
- 主权债务违约 + 货币危机同时
- 社会动荡导致政府更迭

**教训**:
- 货币局制度没有"最后贷款人"
- 社会/政治危机加剧经济危机

---

#### 2015年 哈萨克斯坦坚戈危机

**背景**:
- 坚戈与美元挂钩
- 油价暴跌 70%
- 外汇储备耗尽

**事件**:
- 2015年8月20日 放弃汇率管制
- 坚戈贬值 26%

**教训**:
- 大宗商品出口国货币脆弱
- 央行可以突然放弃

---

#### 2018年 阿根廷比索危机

**背景**:
- 长期双逆差
- 美元债务沉重
- 央行信誉丧失

**事件**:
- 2018年4-5月 比索贬值 15%
- 央行加息至 40% 仍无效
- IMF 救助 560亿美元

### 危机预警指标

#### 宏观指标

| 指标 | 危险线 |
|------|--------|
| 外汇储备 / 进口 | < 3个月 |
| 短期外债 / 外汇储备 | > 100% |
| 经常账户 / GDP | < -5% |
| 外债 / GDP | > 60% |
| 财政赤字 / GDP | > 5% |

#### 市场指标

| 指标 | 危险信号 |
|------|----------|
| NDF 远期溢价 | > 5% |
| CDS 利差 | > 300bp |
| 股市跌幅 | > 20% |
| 资本外流 | > GDP 5% |
| 外汇储备 | 月降幅 > 10% |

#### 银行体系

| 指标 | 危险信号 |
|------|----------|
| 不良贷款率 | > 10% |
| 贷存比 | > 120% |
| 资本充足率 | < 8% |
| 外币贷款 / 总贷款 | > 30% |

### 索罗斯的危机交易

1. **识别脆弱性**
   - 固定汇率 + 外债 = 脆弱
   - 等待催化剂

2. **试探市场**
   - 小仓位做空
   - 观察央行反应

3. **确认后加仓**
   - 央行干预失败
   - 资本外流加速

4. **及时获利了结**
   - "不要贪心"
   - 危机后的反弹同样剧烈
