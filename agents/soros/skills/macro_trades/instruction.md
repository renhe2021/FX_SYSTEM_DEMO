### 能力描述
你能够引用历史上重大的宏观交易案例，分析外汇、利率、信用市场的关键转折点。

- 用户询问某个货币历史走势时
- 用户想了解相似市场环境的历史案例时

### 经典战役库

#### 1985年 广场协议 (Plaza Accord)

**背景**: 1980年代初期，美国贸易赤字严重，美元过度强势
**问题**: 美国制造业遭受重创，日本、德国面临升值压力
**协议**: 五国(G5)央行联合干预美元贬值
**结果**: 
- 日元从 1USD=250日元 升值到 1USD=120日元
- 日本泡沫经济形成
- 出口导向型经济转型

**索罗斯角色**: 早在广场协议前，索罗斯就押注美元贬值，"我早在1985年就开始做空美元"

**教训**: 央行协同干预可以快速改变趋势，但后果可能持续数十年

#### 1997年 亚洲金融危机

**时间**: 1997年7月 - 1998年

**背景**: 
- 泰国、印尼、韩国、马来西亚等存在严重房地产泡沫
- 固定汇率 + 开放资本账户 = 致命组合
- 外债膨胀（主要是短期美元债）

**危机演变**:
1. 泰国外汇储备耗尽，放弃固定汇率
2. 资本外流蔓延至整个东南亚
3. 印尼盾暴跌，社會动荡
4. 韩国面临主权违约风险
5. 香港联系汇率制受冲击

**索罗斯操作**:
- 做空泰铢（Thai Baht）
- 买入韩元看跌期权
- 通过量子基金大量借入泰铢抛售

**教训**:
- 固定汇率是脆弱的，尤其是在外债高企时
- "这次不一样"是最危险的想法
- 流动性枯竭会引发连锁反应

**当前适用**: 新兴市场固定汇率 + 外债上升 + 房地产泡沫

#### 2007-2008 全球金融危机

**背景**: 美国次贷危机蔓延至全球金融体系

**关键事件**:
- 2007年8月: BNP Paribas 冻结赎回
- 2008年3月: Bear Stearns 倒闭
- 2008年9月: Lehman Brothers 破产
- 2008年10月: 全球央行联合降息

**外汇市场反应**:
- 美元 initially 下跌（流动性危机）
- 然后大幅反弹（避险需求）
- 日元升值（避险货币）
- 新兴市场货币暴跌

**索罗斯操作**:
- 大举做空次贷相关产品
- 买入美国国债（避险）
- 在雷曼倒闭前大量持有现金

**教训**:
- 极端风险事件中流动性比什么都重要
- "too big to fail" 意味着政府会干预
- 危机后的反弹往往快速且剧烈

#### 2014年 瑞士央行弃守EUR/CHF下限

**背景**: 
- 瑞士央行自2011年起维持 EUR/CHF 1.20 下限
- 欧洲央行QE预期导致欧元承压
- 瑞士央行外汇储备膨胀失控

**事件**: 2015年1月15日，瑞士央行突然宣布放弃下限

**市场反应**:
- EUR/CHF 从 1.20 瞬间暴跌至 0.85
- 瑞士股市盘中暴跌30%
- 外汇经纪商大规模倒闭（Alpine、FXCM等）

**教训**:
- 央行承诺是不可靠的
- "稳赚不赔"的下限交易可以让你倾家荡产
- 央行可能随时改变政策

---

#### 2015年 人民币811汇改

**背景**: 
- 中国经济放缓，人民币被高估
- 资本外流加剧
- IMF 纳入SDR货币篮子需要市场化

**事件**: 2015年8月11日，央行宣布完善汇率中间价报价

**市场反应**:
- 人民币连续三天大幅贬值
- 全球股市暴跌
- 新兴市场货币承压

**后续演变**:
- 资本管制加强
- 外汇储备消耗近1万亿美元
- 人民币双向波动成为常态

**教训**:
- 人民币不是完全市场化的货币
- 央行政策目标多元化（增长、汇率、资本管制）
- 关注央行的"痛点"

---

#### 2016年 英国退欧 (Brexit)

**背景**: 
- 2016年6月23日英国公投退欧
- 市场普遍预期留欧

**结果**: 退欧派意外获胜

**市场反应**:
- 英镑单日暴跌11%（历史上最大跌幅之一）
- 欧元同时下跌
- 日元、黄金暴涨（避险）
- 英国股市盘中震荡

**索罗斯洞察**:
- 退欧对英国的伤害大于欧盟
- 英镑可能跌至与美元平价
- 预言成真：2022年 GBP/USD 确实短暂触及1.05

### 输出格式
- 先给出当前市场的核心判断
- 引用1-2个最相关的历史战役
- 分析异同点
- 给出交易建议（风险提示）

### 分析框架
1. **基本面**: 汇率估值、央行政策、国际收支
2. **资金流**: 套利交易、资本流动
3. **政治风险**: 选举、政策不确定性
4. **技术面**: 趋势、拥挤度
5. **央行博弈**: 政策空间、干预可能性
